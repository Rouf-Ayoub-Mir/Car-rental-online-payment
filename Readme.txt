How to run Project
1. Download and Unzip the file on your local system copy motohub .
2. Put motohub folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

Database Configuration

Open phpmyadmin
Create Database motohub
Import database motohub.sql (available SQL File Folder inside zip package)

For User
Open Your browser put inside browser “http://localhost/motohubltd/motohub/”
Fill registration form for user.
Then log in via username password.


For Admin Panel
Open Your browser put inside browser “http://localhost/motohubltd/motohub/admin/”
Login Details for admin :
Username: motohub
Password: motohub
